# Jessica Peng 

Hi! This is my personal portfolio website--all about me, my work, and the things I've built!

I'm an **AI Software Engineer at Goldman Sachs** and a **Columbia University CS grad**, working at the intersection of artificial intelligence and investment banking.

## What's on the site

The site is a single-page portfolio split into these sections:

- **Hero / About** — who I am, what I do, and how to reach me
- **Experience** — a timeline of my career, from Goldman Sachs and Apple to research at Columbia, Berkeley, and Harvard Medical School
- **Education** — Columbia University (B.S. in Computer Science) and beyond
- **Skills** — AI/ML, engineering, and finance tools I work with
- **Projects** — things I've built, from a first-responder disaster app to medical AI phantoms, computer vision pipelines, and wearable tech
- **Contact** — a form and links to get in touch

## Tech stack

Built with **React 19** and **Vite**.

```bash
npm install
npm run dev     # local dev server at http://127.0.0.1:5173
npm run build   # production build
```
